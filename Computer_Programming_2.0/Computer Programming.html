Types of programming Languages

1.Low-level programming language

-is machine-dependent(1s and 0s) programming language.
-The processor runs low-level programs directly without the need of a COMPILER or INTERPRETER so Low-level language can be very fast

Two parts of the Low-Level Language

a. Machine language
- "Machine code or subject code"
- easier to read because it is normally displayed in binary of hexadecimal form
- does not need a translator to convert programs

b. Assembly language(ASM)-designed for specific processors. It represents the set of instructions in a symbolic and human- understandable form. Uses the assembler to convert the assemble language to machine language. The advantage of both is that require less memory execution time and program

2. High Level Language Programming -requires compiler or interpreter. Designed for developing user-friendly software programs and websites.
Compilers- translate the high level instructions into 1s and 0s
Interpreters- translate the program line by line as the program is runnin

3.High level language allow programmers to use English-like instructions.

a. Procedural Oriented Programming-is derived from the structured programming and based upon the procedure call concepts. It divides a program into a small procedures called routines and functions.

-used by the software programmer editor like IDE, Adobe Dreamweaver of Microsoft Visual Studio

b. Object Oriented Programming

-based upon the objects

c. Natural Language
